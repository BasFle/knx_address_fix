# Repo fuer die Fragen aus Compnet

## Importieren
`Strg+Shift+I` druecken. Dann file suchen und folgende Optionen auswaehlen im Dropdown `Basic (and reversed card)`.

## Contributen 
Fragen entweder direkt im [CN.txt](./CN.txt) im webeditor aendern. Sonst einfach in anki bearbeiten, exporten und eure Aenderungen committen.
Am besten schreibt ihr mir privat nochmal, dann reagier ich schneller.
